Work

Full-stack projects

Hiking Equipment E-commerce website [Github]

An fullstack E-commerce site that users can browse and purchase hiking equipments. Each customer will create their unique account to purchase items. Integrated a chatbot with openAI Api for smoother customer support process. For payment process, I integrated the Stripe API to handle online payments.


Airbnb Web Scraper[Github]


An app to extract information from Airbnb listings by your given destinations, check-in date, check-out date, guest numbers and targeted total price with sorted total price from low to high. You can also download the data as a csv file for further analysis.


Machine Learning projects

Wine Quality prediction App [Github]

An App to predict the quality of wine based on the given features. The app allows users to input the features of the wine and get the predicted quality of the wine.


Front-end projects

Cake Portfolio Website[Github]

An interactive portfolio to showcase the cakes and pastrie. Users can browse the cakes and pastries and contact the owner for orders.


Android Mobile projects

Tip Calculator App[Github]

An app helps users calculate tips and total amount based on their bill amount. Created a Splitting function to calculate the amount per person with add button and subtraction button.


Data Analysis projects

stocking picking analysis tool[Github]

Using Python and pandas dataframe in this stocking picking analysis tool, I was able to create null hypotheses and provided significant evidence to reject them. In addition, I used several correlation graphs etc to analyze both stock trends and volatility. I provided recommendations to different investors on which stocks is a better investment.
